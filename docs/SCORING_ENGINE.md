# Fanta Score

`deterministic-v1` restituisce un punteggio 0–100 per ruolo, componenti, affidabilità, rischio, tier ed una spiegazione italiana. Le componenti sono minuti attesi, titolarità, rendimento storico, bonus per 90, forza squadra, piazzati e penalità rischio.

Valori mancanti ricevono un baseline neutro, non zero artificiale. Il modello è una baseline dimostrativa, versionata e testata; non è una proiezione statistica né un modello ML.
